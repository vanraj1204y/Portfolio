Sprint Management Tool

Sprint Management Tool is an internal productivity and task management system developed for our own company to streamline sprint planning, task tracking, and employee performance monitoring. The platform provides a centralized view of all tasks assigned to employees along with the time spent on each task.

The system supports role-based access control with multiple user roles such as Developer, QA, HR, and Admin. Each role has specific permissions to manage tasks, view reports, and access sprint-related data. Weekly task planning and daily task execution are managed efficiently within the platform.

At the end of each working day, employees can generate a complete work report with a single click and directly send it to their manager via email. The application also includes a real-time notification system that alerts users whenever a new task is assigned to them.

This project was fully developed using AI-assisted development, where the entire application was created through well-defined prompts without manually writing individual code files. Laravel Vite was used to ensure fast page loading and a smooth user experience.
